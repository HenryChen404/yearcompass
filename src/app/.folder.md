# Folder: /src/app

1. **地位**: Next.js App Router 入口，所有页面路由定义
2. **边界**: 接收 URL 请求，组装组件输出页面
3. **约束**: 页面只做组装，业务逻辑放 /core，UI 组件放 /components

## 成员清单

| 文件/文件夹 | 角色 | 职责 |
|-------------|------|------|
| layout.tsx | Root Layout | 全局布局、字体、Meta 信息 |
| page.tsx | Home Page | 首页三栏布局（目标栏 + 日历 + 任务栏） |
| globals.css | Design System | 全局样式变量、印象派配色方案 |
| trends/ | Route | 趋势图展示页面，历史进度可视化 |
