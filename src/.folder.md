# Folder: /src

1. **地位**: 源代码主目录，包含所有应用逻辑
2. **边界**: 接收用户交互，输出完整的 Web 应用
3. **约束**: 遵循 Next.js App Router 规范，组件保持职责单一

## 成员清单

| 文件/文件夹 | 角色 | 职责 |
|-------------|------|------|
| app/ | Route Handler | Next.js App Router 页面和路由 |
| components/ | UI Layer | 可复用 React 组件 |
| core/ | Business Logic | 核心业务数据和规则 |
| lib/ | Infrastructure | 工具函数和外部服务封装 |
| types/ | Type Contracts | TypeScript 类型定义 |
